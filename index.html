Prework Study Guide

HTML
  • The h2-6 element represents the size of the text
  • The head element contains information about the webpage
  • The body element represents the visible content showed to the user
CSS
  • A margin indicates how much space we want around the outside of an element and is not visible on the page
  • Padding adds space around the content inside an element
  • The Asterik(*) symbol declares that all elements on the page will have the same styling
  • Px stands for pixels and it is used to determine the height or width of the element
  • The HTML link element is used to connect an external css style sheet to your current project
GIT
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add moves changes from working directory to the staging
  • git config allows you to specify the username and email that is used with your commits
JAVASCRIPT
  • Each line of code in JavaScript must end with a semi-colon
  • The if statement is used to evaluate if the condition provided is true or false
  • The for loop is used to log multiple conditions
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
